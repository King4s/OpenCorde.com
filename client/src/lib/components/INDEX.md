# /client/src/lib/components/

Purpose: Reusable Svelte UI components.

Pattern: PascalCase.svelte. Organized by feature area.

| Directory | Purpose |
|-----------|---------|
| layout/ | App shell: Header, Sidebar, Navigation |
| chat/ | Message list, input box, reactions, embeds |
| voice/ | Voice controls, participant list, status |
| modals/ | Dialogs: Create server, invite user, settings |
| settings/ | User preferences: theme, audio, privacy |
